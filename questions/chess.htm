Calculate number of positions a Queen can attack on a chessboard

You will be given a square chess board with one queen and a number of obstacles placed on it. Determine maximum number of squares the queen can attack in all the possible moves.

Requirements

  • Expose a POST method with endpoint /chessgame
  • Return integer result as json

Constraints

  • Input will be JSON object with Queen (K) and obstacle (X) positions specified in an array.
  • n is an integer specifying square shaped chessboard size. 1 <= n <= 200

Input description

A queen is standing on n * n chessboard. The chess board's rows are numbered from 1 to n, going from bottom to top. Its columns are numbered from 1 to n, going from left to right.

The HTTP POST request will come with a body of Content-Type: application/json

Sample input JSON

In below example, 5 is the chessboard size and 3 obstacles are present on it. Queen is standing at square (1,2) and obstacles are present at (3,1), (3,2) and (4,3)

    [
      [
        "","K","","",""
      ],
      [
        "","","","",""
      ],
      [
        "X","X","","",""
      ],
      [
        "","","X","",""
      ],
      [
        "","","","",""
      ] 
    ]

Sample output

Number of squares where Queen can attack is 9.

9
